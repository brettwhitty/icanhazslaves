# inspect

## Description
The `inspect` action type involves passive discovery and investigation of the project environment without modifying any state or infrastructure.

## Scope
- Listing directory contents.
- Reading file contents.
- Searching for patterns or files.
- Retrieving external documentation or URL content.
- Viewing code outlines or specific code items.

## Allowed Tools (Heuristic)
- `list_dir`
- `view_file`
- `view_file_outline`
- `view_code_item`
- `grep_search`
- `find_by_name`
- `read_url_content`
- `search_web`
- `read_terminal`
- `command_status` (for non-destructive commands)
